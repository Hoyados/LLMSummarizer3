# LLMSummarizer3

LLMSummarizer3 は、指定した URL の内容を要約する iOS アプリです。
ブラウジングした情報を元に、要点を短く整理して表示することを目的としています。

## 主な機能

- URL を入力して要約を生成
- 要約結果の表示と確認

## 動作環境

- Xcode 15 以降
- iOS 17 以降を想定

## セットアップ / デプロイ

1. 本リポジトリをクローンします。
2. `LLMSummarizer3.xcodeproj` を Xcode で開きます。
3. 署名設定を行い、対象デバイスまたはシミュレータで実行します。

> **重要**
> デプロイはご自身で Xcode を使用して実施する必要があります。
